# youtube_cli_dl
Downloads Youtube videos in audio or video format. For practice.

## Installation

### Dependencies
- python, pip, pytube
#### On Arch based distros:
```
sudo pacman -S python python-pip
```
Then:
```
pip install pytube
```

### Download
```
git clone https://github.com/jamesyoung-15/youtube_cli_dl
```
## Usage
Run script:
```
cd youtube_cli_dl/
python youtube_dl.py
```
